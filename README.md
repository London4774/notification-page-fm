# Frontend Mentor - Notifications Page Solution

## 🛠️ Технологии
- **React** (Функциональные компоненты, Hooks)
- **Tailwind CSS** (Mobile-first дизайн, верстка)
- **Vite** (Сборка проекта)
- **Vercel** (Деплой)

  ## 🚀 Демонстрация
[Посмотреть проект вживую (Live Site)](notification-page-fm-lyart.vercel.app)


## 📦 Как запустить проект локально

1. Склонируйте репозиторий:
   git clone (этот репо)

2. Установите зависимости:
  npm install

3. Запустите сервер для разработки:
   npm run dev
